A Free and Open Source OAuth 2.0

An OAuth 2.0 server based on Skoruba's IdentityServer4 with some tweaks.

Go to the Server Dashboard

Features

Free to use

Everyone can sign-up to the service, register a client, and use it as identity provider.

Open Source

The project is entirely Open Source, the code is available in GitHub.

ASP.NET Core 3.1

The project is written in C# using the latest version of the open source framework: ASP.NET Core 3.1.

Based on Open Source software

The project uses IdentityServer4.Admin as base framework.

Full documented REST API.

The endpoint can be tested on the Swagger platform.
